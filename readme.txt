INSTRUCTIONS ON HOW TO RUN THE GAMEIn other to run the game, please follow the following steps1. Make sure python is installed.2. Extract the zip file containing necessary scripts.3. Open command prompt and navigate to the folders directory4. Install the required dependencies by running the following commando pip install -r requirements.txt5. Once the dependencies are installed run the game by running the following commando python main.py6. The game window should open and you can start playing.
